# Propuesta comercial — Ferretería Kozlowzki

Landing page de propuesta comercial B2B para **Ferretería Kozlowzki** (Jardín América + Puerto Rico). Presenta planes de tienda online, integración con sistema de gestión, precios por etapas y requisitos del cliente.

## Contenido

- Problema y solución para ferreterías con catálogo grande (500+ ítems)
- Modalidades de implementación (Básico / Premium / Gold)
- Alcance técnico: catálogo, stock, Mercado Pago, envíos
- Precios de referencia y etapas de desarrollo
- Requisitos del cliente (fotos, dominio, ERP, logística)

## Tecnologías

- HTML monolítico
- Bootstrap 5 + Bootstrap Icons
- Google Fonts (Plus Jakarta Sans)
- Sin backend — documento estático para compartir con el cliente
